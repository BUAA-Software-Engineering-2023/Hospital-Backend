# 用例描述
```
用例名称： 查看病历
用例编号： UC11
用例简介： 描述了用户查看病人病历的过程
参与者：用户
触发条件：用户进入了查看病历的页面
前置条件：医生完成了对患者的诊断
基本流程：
	(系统)1.系统进入查询模式
	(用户)2.向系统输入自己的信息
	(系统)3.系统根据输入的信息，向用户展示病历的信息。
扩展流程：
	(可选) 1-3a. 管理员可以在任何时间选择退出系统。
  	(系统)1. 系统进入管理员模式。
    (系统)2. 管理员执行退出操作。
    (用户)3. 系统退出，保存当前信息。
结论：用户得到了病历的信息
后置条件：系统展示了用户的病历
补充约束：系统响应迅速，登录响应在1s之内。
```
